General Information

Web-based data repositories represent a constantly updating source of information. Most of these information sources have built-in methods/APIs designed for requesting and submitting data in a streamlined format. This workshop aims to introduce you to the basic of HTTP api usage for database such as NCBI, UniProt, and The Human Protein Atlas using the Python package "requests" and parsing of html/xml file using "BeautifulSoup4".

Who: This session is directed at intermediate to advance Python users wanting to learn about automating of data extraction from publicly available repository using the default packages coming with an Anaconda Python 3 installation.
